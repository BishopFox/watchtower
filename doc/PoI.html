    new(data) click to toggle source

    Initializes the Poi (Point of Interest) object. Accepts a hash containing the following keys:

    :file_type

    The file type in which the PoI was found

    :file

    The file in which the PoI was found

    :line_number

    The line number at which the PoI was found

    :match

    The specified signature which this PoI matches

    :snippet

    A code snippet of the match in context

    :group

    The group to witch the signature belongs

    Usage:

    data = {
        :file_type   => 'php',
        :file        => 'example.php',
        :line_number => 100,
        :match       => 'exec(',
        :snippet     => 'exec(evilfunction())',
        :group       => 'dangerous_functions',
}
poi = PoI.new(data)

def colorize
        @file                  = @file.colorize(@@color_file)
        @line_number = @line_number.colorize(@@color_line_number)
        @snippet       = @snippet.sub(@match, @match.colorize(:background => @@color_match))
end
